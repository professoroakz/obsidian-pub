# Modules Directory

This directory contains git submodules for external resources and dependencies.

## Submodules

### obsidian-plugins
Official Obsidian releases and community plugins repository.

### awesome-obsidian
Curated collection of awesome Obsidian resources.

## Usage

See [docs/SUBMODULES.md](../docs/SUBMODULES.md) for detailed instructions.

To initialize all submodules:

```bash
make submodule-init
```
